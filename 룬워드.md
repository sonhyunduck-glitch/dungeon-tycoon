# 룬워드 조합표 (소켓 시스템) — 전체 28종

발동 조건: **소켓 수 = 룬 개수**가 정확히 일치하는 장비 + **부위(무기/방어구) 일치** + 필요 룬(**순서 무관**) 전부 장착. 데이터: [js/data.js](js/data.js) `G.DATA.RUNEWORDS`.

- 🗡️ **무기 룬워드** → 무기(검·도끼·단검) 소켓 베이스
- 🛡️ **방어구 룬워드** → 투구·갑옷·장갑·신발 소켓 베이스 (반지·목걸이는 소켓 없음)
- 완성 시 아이템 이름이 룬워드명으로 바뀜(청록색). **🔓공개**=처음부터 레시피 제공 / 나머지=발견형. **150층 돌파 시 전체 룬워드 레시피 공개**(`G.DATA.RUNEWORD_OPEN_FLOOR`)
- ⚙️ **표 수치는 기준값**. 완성 순간 스탯별로 **기준의 85%~120%** 폭에서 1회 굴려 영구 고정 → 같은 룬워드라도 **상위 옵션(품질 ~120%)** 가능. 품질%는 장비/소켓 정보에 표시. (폭 조절: [js/item.js](js/item.js) `G.runeword.VAR_LO/VAR_HI`)
- 집계: **무기 14 + 방어구 14 = 28** (3룬 14 · 4룬 8 · 5룬 6)
